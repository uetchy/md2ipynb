# md2ipynb

Make your markdown files interactive.

## Usage

```node
var md2ipynb = require("md2ipynb");
var ipynb = md2ipynb(markdownDocument);
console.log(ipynb);
```

## Install

```
$ npm install md2ipynb
```

## Build

```
$ git clone https://github.com/uetchy/md2ipynb
$ cd md2ipynb
$ npm install
$ npm test
```
